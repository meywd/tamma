# Epic 1: Foundation & Core Infrastructure

This directory contains all stories and context files for Epic 1, which establishes the foundational infrastructure for the Tamma platform.

## Epic Overview

**Goal**: Establish multi-provider AI abstraction, multi-platform Git integration, and hybrid orchestrator/worker architecture.
